# YSTI 原神人格类型指标测试 v2.0

> Yuanshen Type Indicator — 你是提瓦特的谁？

基于 10 个人格维度、20 道测试题，精准匹配 32 位原神角色中最接近你灵魂的那一位。

## Tech Stack
- Vite + React + TypeScript
- Vitest for testing
- GitHub Pages / Cloudflare Pages deployment

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Test

npm test
